#  Responsive Sidebar Menu for PhotoShopy Website

A modern and responsive sidebar navigation menu designed for a photography website called **PhotoShopy**.  
This project is built using **pure HTML and CSS**, focusing on clean UI design, smooth animations, and responsive behavior without using JavaScript.

---

##  Features

-  Fully responsive sidebar layout
-  Pure HTML & CSS implementation (no JavaScript)
-  Hamburger menu toggle using checkbox technique
-  Modern UI with glassmorphism-inspired design
-  Smooth hover effects and transitions
-  Responsive background image handling
-  Social media icons with interactive hover animations
-  Font Awesome icons and Google Fonts integration

---

##  Technologies Used

- **HTML5**
- **CSS3**
- **Font Awesome**
- **Google Fonts**
